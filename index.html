PDF Slideshow

Keyboard Shortcuts

  1. H - Show/Hide shortcut keys
  2. O - Open file dialogue to choose pdf file
  3. ArrowUp - Increase page change interval by 100ms and start slideshow
  4. ArrowDown - Decrease page change interval by 100ms and start slideshow
  5. ArrowLeft - Show previous page
  6. ArrowRight - Show next page
  7. R - Reload current page
  8. Space - Start/Stop slideshow
  9. F or F11 - Toggle full screen mode
  10. T - Toggle light/dark theme
